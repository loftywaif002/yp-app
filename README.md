### Instructions
1. git clone https://github.com/loftywaif002/yp-app.git
2. cd yp-app
3. npm install
4. npm run start
5. App should be running on http://localhost:5000


#Technology User
 FrontEnd: React.js  16.5.2
 Backend: Node v8.11.1 (express)
 Database: postgres 9.6
 ORM: Sequelize 4.39.1
